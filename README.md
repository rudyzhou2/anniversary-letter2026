# 💍 Anniversary Letter 2026

A private, password-protected anniversary letter for Sandy.

## Setup

Add your photos to the `images/` folder named exactly:

- `photo1.jpg` through `photo10.jpg`

## Password

Default password: `sandy`
To change it, edit `const PASSWORD = 'sandy';` in `index.html`.

## GitHub Pages

Go to **Settings → Pages → Branch: main → / (root)** and save.
Your letter will be live at: `https://rudyzhou2.github.io/anniversary-letter2026`
