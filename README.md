# 💍 Anniversary Letter 2026

A private, password-protected anniversary letter for Sandy.

## Setup

1. Add your photos to the `images/` folder named exactly:
   - `photo1.jpg`
   - `photo2.jpg`
   - `photo3.jpg`
   - `photo4.jpg`
   - `photo5.jpg`
   - `photo6.jpg`

2. Open `index.html` in a browser — or host it via GitHub Pages.

## Password

Default password: `sandy`
To change it, edit line `const PASSWORD = 'sandy';` in `index.html`.

## GitHub Pages

Go to **Settings → Pages → Branch: main → / (root)** and save.
Your letter will be live at: `https://rudyzhou2.github.io/anniversary-letter2026`
